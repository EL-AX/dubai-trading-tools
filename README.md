## ðŸ“ˆ Indicators Explained

### RSI (Relative Strength Index)
- **Overbought** (>70) = Potential SELL
- **Oversold** (<30) = Potential BUY
- **Normal** (30-70) = Monitor

### MACD (Momentum)
- **Bullish Crossover** = BUY signal
- **Bearish Crossover** = SELL signal
- **Divergence** = Strongest signals

### Bollinger Bands (Volatility)
- **Touch upper band** = Resistance
- **Touch lower band** = Support
- **Band expansion** = High volatility incoming
